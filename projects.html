Indigenous Art Gallery API - University Project

My high distinction project for the unit SIT331 Secure Backend Development. The project is a RESTful API that allows users to interact with an art gallery database. The API is built using Node.js, Express.js, and MongoDB. It also has a simple user interface build using Vue.js. The API is secured using JWT tokens and has a range of features including user authentication and CRUD operations on the database.
